# E-commerce Testing Project (QE)

## Overview
This project demonstrates end-to-end Quality Engineering practices including manual testing, API testing, and defect reporting.

## Modules Covered
- Login
- Search
- Cart
- Checkout
- API Testing

## Testing Types
- Functional Testing
- Negative Testing
- API Testing (Postman)
- Exploratory Testing

## Tools Used
- Postman
- GitHub
- Excel

## Key Highlights
- Created 15+ detailed test cases
- Performed API testing using ReqRes public APIs
- Identified and documented bugs with severity and priority
- Validated responses and status codes
